## develop

```sh
$ npm install
$ gulp build
$ gulp watch
```

chromeの拡張機能の管理ページで「デベロッパー モード」にチェックを入れ、「パッケージ化されていない拡張機能を読み込む...」で `build/` ディレクトリを読み込むことで動作確認ができます。（`build/` ディレクトリは `gulp build` コマンドの実行で生成されます）
